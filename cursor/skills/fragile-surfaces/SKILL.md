# Fragile Surfaces

        ## Quando usar
        Quando a mudança toca autenticação, dados, migrações, sincronização, infra ou billing.

        ## Saída
        - arquivo mais sensível
        - menor blast radius
        - testes obrigatórios
        - plano de rollback
        - observabilidade exigida

        ## Regra
        Quanto mais frágil a superfície, menor deve ser o lote de mudança.
