# NSC — English Documentation

## Objective

NSC (NonSlopContext) is a contextual governance framework for autonomous coding agents.

Its objective is to reduce:

- AI slop
- architectural entropy
- unsafe refactors
- excessive autonomy
- context degradation
- destructive large-scale modifications

---

## What NSC does NOT do

NSC does not completely eliminate:

- mistakes
- hallucinations
- bugs
- poor decisions
- implicit contract breakage

Human supervision is still required.

---

## Philosophy

NSC is not focused on increasing intelligence.

NSC focuses on:

- persistent context
- operational predictability
- governance
- incremental execution
- blast radius reduction

---

## Installation

### NPX

```bash
npx nsc
```

### Global

```bash
npm install -g @nsc/core
```

Then:

```bash
nsc
```
