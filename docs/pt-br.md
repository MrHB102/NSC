# NSC — Documentação PT-BR

## Objetivo

O NSC (NonSlopContext) é um framework de governança contextual para agentes autônomos de código.

Seu objetivo é reduzir:

- AI slop
- entropia arquitetural
- refactors perigosos
- autonomia excessiva
- degradação de contexto
- modificações destrutivas em larga escala

---

## O que o NSC NÃO faz

O NSC não elimina completamente:

- erros
- alucinações
- bugs
- decisões ruins
- quebra de contratos implícitos

Supervisão humana ainda é necessária.

---

## Filosofia

O foco do NSC não é aumentar inteligência.

O foco é:

- contexto persistente
- previsibilidade operacional
- governança
- execução incremental
- redução de blast radius

---

## Instalação

### NPX

```bash
npx nsc
```

### Global

```bash
npm install -g @nsc/core
```

Depois:

```bash
nsc
```
