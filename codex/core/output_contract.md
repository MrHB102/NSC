# Output Contract

Todo agente deve responder com:

1. objetivo entendido
2. contexto usado
3. hipótese principal
4. plano de mudança
5. riscos
6. arquivos afetados
7. validação
8. próximos passos

Para mudanças grandes, exija sempre:
- resumo executivo
- diff mental
- risco residual
- pedido de aprovação explícito
