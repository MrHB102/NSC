# System Map

        ## Quando usar
        Sempre que a base tiver múltiplos serviços, integrações ou infraestrutura híbrida.

        ## Saída mínima
        - serviços principais
        - contratos entre serviços
        - filas e jobs
        - bancos e stores
        - integrações externas
        - pontos de acoplamento
        - pontos de falha

        ## Regra
        Se o agente não consegue desenhar o mapa, ele ainda não entendeu o sistema.
