# MetaContext OS — Agent Pack

## Objetivo

Este pack entrega contexto persistente, estruturado e auditável para agentes de código em bases grandes e frágeis.

## Como preencher antes de usar

1. Atualize `AGENTS.md` ou o arquivo nativo do agente.
2. Complete os arquivos de `core/`.
3. Ajuste os prompts de `prompts/` para o seu sistema.
4. Revise os skills em `skills/`.
5. Só então permita mudanças grandes.

## Política de operação

- primeiro entender
- depois mapear
- depois planejar
- só então editar
- sempre verificar
- nunca esconder risco
