# Legacy Constraints

Liste limitações que não podem ser ignoradas:

- padrões antigos que precisam ser preservados
- APIs públicas congeladas
- migrações pendentes
- dados sujos
- dependências desatualizadas
- coisas que parecem erradas mas são comportamento esperado do legado

Sempre documente:
- o que não mexer
- o que mexer com cautela
- o que é candidato a remoção futura
