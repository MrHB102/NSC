# Implicit Rules Mining

        ## Quando usar
        Quando algo "sempre foi assim" e ninguém sabe explicar o motivo.

        ## Perguntas
        - qual comportamento é esperado?
        - qual exceção é normal?
        - quem conhece essa regra?
        - existe contrato externo implícito?
        - qual teste comprova isso?

        ## Saída
        Converta as regras implícitas em itens explícitos no contexto.
