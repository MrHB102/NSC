# Tradeoff Analysis

        ## Quando usar
        Toda vez que existir mais de uma solução razoável.

        ## Estrutura
        Para cada alternativa:
        - benefício principal
        - custo principal
        - risco principal
        - impacto em manutenção
        - impacto em prazo
        - impacto em confiabilidade

        ## Regra
        Não escolha a solução "bonita" sem justificar o custo operacional.
